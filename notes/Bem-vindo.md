# 🌟 Bem-vindo à sua Wiki Pessoal!

Esta é uma aplicação local autohospedada para organizar seus pensamentos, documentar seus estudos em **Análise e Desenvolvimento de Sistemas (ADS)** e gerenciar suas anotações em arquivos **Markdown** simples.

## 🚀 Recursos Principais:
- **Organização Local:** Suas notas são salvas em arquivos `.md` reais na pasta `notes/` no seu HD. Você é dono dos seus dados!
- **Visualização em Tempo Real:** Edite o texto na aba "Editor" e veja a renderização instantânea do Markdown formatado na aba "Preview".
- **Busca Global:** Use a barra de pesquisa na lateral para buscar termos nos títulos ou no conteúdo das notas.
- **Formatação Rica:** Suporte a cabeçalhos, listas, blocos de código com realce, tabelas, links e citações.

## 💡 Dicas de Markdown:

Você pode estruturar suas anotações facilmente usando símbolos simples:

### Títulos:
Use `#` para título principal, `##` para subtítulo e `###` para tópicos.

### Estilo de Texto:
- Use **negrito** colocando o texto entre asteriscos duplos: `**texto**`.
- Use *itálico* colocando o texto entre asteriscos simples: `*texto*`.
- Use `código em linha` colocando o texto entre crases: `` `código` ``.

### Listas:
Use `-` ou `*` para criar listas de tópicos:
- Item de tópicos 1
- Item de tópicos 2

Ou números para listas ordenadas:
1. Primeiro passo
2. Segundo passo

### Blocos de Código:
Use três crases para envolver blocos de códigos em qualquer linguagem:
```javascript
const saudar = (nome) => {
  console.log(`Olá, ${nome}!`);
};
saudar("Gabriel");
```
