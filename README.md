### 🚀 O que é o VortexWiki?

O **VortexWiki** é um organizador de notas pessoal *local-first* desenvolvido para rodar localmente na sua máquina. Ele permite que você crie, edite, exclua e pesquise anotações em formato **Markdown** através de uma interface web minimalista, moderna e com estética cyberpunk.

Ideal para estudantes e desenvolvedores documentarem seus estudos, comandos úteis ou ideias rápidas.

---

### ✨ Funcionalidades Principais

- **Armazenamento Local:** Suas notas são salvas em arquivos `.md` reais na pasta `notes/` do seu computador. Você tem total controle e propriedade sobre seus dados.
- **Auto-Save Inteligente:** O sistema salva as notas automaticamente 1 segundo após você parar de digitar, garantindo que você nunca perca seu trabalho.
- **Editor & Visualização (Preview):** Alterne facilmente entre a aba "Editor" (com fonte mono) e a aba "Preview" que renderiza o Markdown compilado com tipografia rica e blocos de código formatados.
- **Mecanismo de Busca Global:** Barra de busca que realiza buscas de texto completo (no título e dentro do conteúdo das notas) com destaque para resultados mais relevantes.
- **Visual Premium:** Estética dark e efeitos de vidro (glassmorphism) com brilhos ambientais e ícones dinâmicos.

---

### 🛠️ Tecnologias Utilizadas

- **Servidor (Backend):** Node.js, Express (para rotas de arquivos e pesquisa)
- **Compilador Markdown:** `marked.js` (renderização client-side leve e rápida)
- **Interface (Frontend):** HTML5, CSS3 (Vanilla com HSL colors), JavaScript ES6, Lucide Icons

---

### ⚙️ Como Instalar e Rodar Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/Vortexyweb/personal-wiki.git
   cd personal-wiki
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie a Wiki:
   ```bash
   npm start
   ```

4. Acesse no seu navegador:
   👉 **`http://localhost:3000`**
